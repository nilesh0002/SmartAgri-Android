# SmartAgri-Android 🌾

A comprehensive and Excellent Android application for modern farming, built with **Kotlin** and **XML Fragments**. SmartAgri is designed to empower farmers with real-time market data, tailored crop recommendations, and intelligent farming insights right at their fingertips.

## 🚀 Features

*   **🌱 Crop Calendar:** Localized crop schedules and personalized planting guidance to maximize yield.
*   **☀️ Weather Forecast:** Real-time weather conditions and forecasts to help plan farming activities.
*   **🐛 Pest Information:** Detailed database of common agricultural pests along with actionable solutions and prevention techniques.
*   **🌍 Soil Information:** Guides on different soil types and which crops thrive best in each environment.
*   **📈 Live Market Prices (Mandi):** Real-time commodity pricing with detailed charts to help farmers get the best value for their produce.
*   **🤖 AI-Powered Fertilizer Recommendations:** Intelligent suggestions for fertilizer usage based on crop and soil needs, powered by **Gemini AI**.
*   **🏛️ Government Schemes:** Information about various government schemes and subsidies available for farmers.
*   **🌐 Multi-language Support:** Accessible in multiple languages for a wider range of farmers.
*   **📞 Helpline & Support:** Direct access to helplines and support services.
*   **🔐 Secure Authentication:** Seamless Login and Registration flow styled with modern Material 3 inputs.
*   **✨ Beautiful, Modern UI:** Completely overhauled with **Material Design 3**. Features dynamic `MaterialCardView` layouts, sleek rounded components, improved typography, and responsive, accessible interfaces for farmers.

## 🛠️ Tech Stack

*   **Language:** Kotlin
*   **UI Toolkit:** XML Layouts & Material Design 3 Components (MaterialCardView, TextInputLayout, BottomNavigationView)
*   **Architecture:** MVVM (Model-View-ViewModel)
*   **Navigation:** Android Jetpack Navigation Component
*   **Networking:** Retrofit & OkHttp
*   **AI Integration:** Google Gemini AI SDK for personalized recommendations
*   **Local Database:** Room Database
